MOXI CORP LOGO COLOUR THEME

Palette:
- Cyan: #04DBFA
- Electric blue: #2B9AF4
- Indigo: #5D67F2
- Violet: #9B32E9
- Magenta: #DB0BD9
- Background: #030306

HOW TO USE:
1. Upload brand-theme.css into the same folder as styles.css.
2. Replace your existing HTML pages with the pages in this package.
3. Keep your existing styles.css, script.js, images folder, logo files and Stripe links.
4. The support email remains support@moxicorpstudio.com.au.
5. Formspree remains connected to https://formspree.io/f/xvzjprdv.

The new stylesheet loads after the existing styles and changes the site-wide colour system without removing page content, forms, links, screenshots or JavaScript hooks.
