# Moxi Corp Website

Production-ready static website for Moxi Corp.

## Main files

- `index.html` — home page
- `services.html` — website services
- `portfolio.html` — selected work
- `pricing.html` — package pricing and comparison
- `about.html` — business story and values
- `process.html` — the website build process
- `faq.html` — interactive FAQs
- `contact.html` — Formspree quote form
- `ecommerce.html` — eCommerce service page
- `addons.html` — optional extras
- `privacy-policy.html` — privacy policy
- `thanks.html` — form confirmation page
- `404.html` — custom not-found page
- `styles.css` — shared design system and responsive styling
- `script.js` — navigation, reveal effects, FAQs and form progress

## Deployment

Replace the existing website files with everything in this folder. Upload every file and the complete `assets` folder to the root of the GitHub repository. Do not upload an old `.git`, `landing page`, `api`, or duplicate `images` folder.

The included `CNAME` points GitHub Pages to:

`www.moxicorpstudio.com.au`

The included Google verification file and meta tag should remain unchanged.

## Contact form

The quote form submits to the existing Formspree endpoint:

`https://formspree.io/f/xvzjprdv`

After a successful submission it redirects to:

`https://www.moxicorpstudio.com.au/thanks.html`
